## 编程

- 不要一边看视频一边运行代码

  缺乏对代码的宏观思考

- 讲解代码时记笔记

  尽量不要暂停, 可以考验自己快速提取信息的能力

- 根据笔记敲代码

  笔记是简介的, 考验根据重点笔记敲代码的能力

  组织模型框架也有深入的了解

  碰到不会的, 可以发现自己的盲区

- 可以对着`代码`部分进行编程
