### 作业-1

- 目的

  一致性检验是针对正互反矩阵的检验

- 思考

  属于代码的边界检验

  - 是否是方阵
  - 是否满足$a_{ij} \times a_{ji} = 1$
  - 是否满足$a_{ij} > 0$

#### 代码实现

[](hw1.m ":include :type=code matlab")

#### 代码技巧

- 通过转置矩阵, 巧妙检验$a_{ij} \times a_{ji} = 1$
- 注意容易产生的 bug

  `A .* (A') ~= ones(mat_r)`返回的不是 true 或 false, 而是布尔矩阵, 只要里面有相同的,布尔矩阵就会有 1, 就不是 false

  - 方法:`sum(sum(A' .* A ~= ones(n))) > 0`或者用`find`
